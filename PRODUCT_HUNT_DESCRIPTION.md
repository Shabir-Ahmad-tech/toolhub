## Full Description (Supports Markdown)

### What is KRUMB.DEV?

KRUMB.DEV is a private-by-design collection of **21 free developer tools** that run entirely in your browser. Nothing you paste, encode, decode, or generate ever leaves your machine.

Built by a developer for developers — no VC bloat, no signup walls, no "enter your email to continue." Just tools that work, formatted in a terminal-inspired UI that feels like home.

### What's Included

**FORMAT & VALIDATE**
- JSON Formatter + Schema Validator (Draft 07 and 2020-12)
- Code Formatter (JavaScript, TypeScript, Python, HTML, CSS, JSON)
- SQL Formatter
- YAML ↔ JSON Converter
- JSON to CSV / CSV to JSON

**DEBUG & INSPECT**
- JWT Decoder (header, payload, expiry countdown)
- Regex Tester (real-time matching, syntax tree explainer, cheat sheet)
- API Response Validator (status + headers table)
- Diff Checker (side-by-side, ignore-whitespace)
- Webhook Tester (Stripe, GitHub, Shopify mock payloads)

**ENCODE & GENERATE**
- Base64 Encoder/Decoder (text + file upload, 2MB limit)
- URL Encoder/Decoder
- UUID Generator (v1/v3/v4/v5, bulk)
- Hash Generator (MD5, SHA-1/256/512, HMAC, batch)
- Password Generator (strength meter, entropy analysis)
- QR Code Generator (custom size, transparent background)

**CONVERT & BUILD**
- Binary/Hex/Decimal/Octal Converter
- Hex to RGB/HSL/OKLCH + Contrast Checker
- cURL → Fetch, Axios, Python, Go
- Cron Expression Builder (visual, human-readable translation)
- .gitignore Generator (17 combinable templates)
- CSS Gradient Generator
- CSS Box Shadow Generator

### Why No Signup?

Every tool uses **client-side JavaScript** — nothing is sent to a server. That means:
- Zero data leakage
- No accounts to manage
- Works offline after first load
- No rate limits
- GDPR-compliant by design (there's no data to collect)

### Tech Stack

- **Framework:** Next.js 16 (App Router, Turbopack)
- **Deployment:** Vercel (SSG, 62 static pages)
- **UI:** Terminal-inspired design, dark mode
- **Auth:** None needed (zero-server architecture)
- **Database:** None (everything runs in the browser)

### Who Is It For?

- **Web developers** debugging JWTs, testing regex, formatting JSON
- **Backend engineers** converting cURL commands, validating API responses
- **DevOps engineers** building cron schedules, generating .gitignore files
- **CS students** learning binary conversion, regex patterns, number bases
- **Anyone** who needs a dev utility without creating another account

### What's Coming

- SQL Formatter (MySQL, Postgres, SQLite dialects)
- Mermaid.js diagram editor
- HTTP Status Codes reference
- HTML to Markdown converter
- And more — all free, all client-side
